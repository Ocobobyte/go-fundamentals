# Intro

## Go

- Focus on simplicity, clarity & scalability
- High performance & concurrency
- Built-in features
- Statically-typed (each statement is checked at compile time)
- Compiled

```go

import "fmt"

  fmt.Print("Hello Ibagophers!")
}
```
